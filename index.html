Conversion Table

Sometimes recipes will give you measurements, but your spoons and cups are in a different measurement. This table may help you with converting teaspoons to milliliters.

1 teaspoon is equal to 4.929 Milliliters, so you can calculate the conversion by multiplying the number of teaspoons by 4.929, as shown below.

Other Website Review

For my website review, I searched for another website that would have some cooking information authored by a person also named Lina...and I actually found someone! She has a recipe blog that has a ton of different recipes posted.

I looked at the source code for this website, and I don't know if I'm qualified enough to make a statement about whether the site was good or bad...but there were a few things that stuck out to me that may have swung the dial toward the "bad" side.
I noticed that the code for the header of the page was incredibly long, and it looked very repetitious. I think maybe that could be cleaned up a little bit to be fewer lines of code (but I don't know enough to really say that definitively).
I also noticed that they used the <div> tag, instead of a <section> tag to separate blog posts, which I thought was odd; in my readings when I was trying to figure out the difference, blog posts were an example of using section tags instead of div tags, so it would have been a good application in this case.
I did see that they used a lot of attributes in their tags though, so that was very good.
